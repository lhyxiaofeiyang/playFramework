Play! 一个颠覆JavaEE规范的web开发框架

SNA（无共享架构）的HTTP session实现机制，让你的大规模集群应用不必担心Session的问题：Play!还提供了ORM等等web开发当中一切所需要的功能，极大程度上提高开发效率。Play!的架构和设计思想是全新的，在Java web快速开发领域秒杀任何竞争对手，直接和Rails竞争。为何Play可以做到那么多其他Java框架做不到的事情呢？秘密就是在于：Play!无视JavaEE陈旧的规范，直接废弃了被奉为天条的Java Servlet规范，抛弃了Java程序员必须遵守的WEB-INF目录结构和WAR打包机制，独辟蹊径，自己编写了完整的HTTP Server和完整的Web框架，这样才造就了新一代的java框架Play!
从Play!身上我们也可以学到一点，那就是不断地挑战和否定过去，才能创想和创造未来，Java社区也只有不断地抛弃陈旧的枷锁，才能开创更美好的未来。

范凯 现任ITEye CEO（原JavaEye创始人） 开源软件的积极推动者和领导人#

把 Play! 称为框架事实上有点小看它了。这是一个全新的 Java Web 应用开发平台、full-stack 的解决方案。从模板到Ajax，从MVC到ORM，从缓存再到定时作业，Web应用所需要的各个层次内容都可以在 Play! 框架中找到对应的实现方法。但是当你决定使用 Play!之前，请抛弃你对 Java 框架所有的认识，你需要保留的只是普通的Java编程水平和基本的HTTP概念。因为 Play!颠覆了J2EE 标准，用超乎简单的代码构建完美的 Web 应用。当然，如果你已熟悉 Ruby on Rails 框架那再好不过了。本书用非常简单易懂的方式介绍 Play!基本模块在实际中的应用，有了它的帮助，事半功倍！

刘漠 OSChina.NET（开源中国社区） 创始人 国内开源潮流的推动者#
